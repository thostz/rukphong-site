💬 คำสั่ง LINE Bot 8 คำสั่ง ▼
จ่าย [จำนวน] [หมวด] บันทึกรายจ่าย — ระบุจำนวนเงินและหมวดหมู่ เพิ่มเข้า Google Sheet ทันที จ่าย 150 อาหาร
ใช้ [จำนวน] [หมวด] เหมือน จ่าย — ใช้แทนกันได้ ใช้ 500 ช้อปปิ้ง
งาน [ชื่องาน] เพิ่ม Task ใหม่ — Status เริ่มต้นเป็น To Do, Priority: Medium งาน ประชุม client พรุ่งนี้
โน้ต [ข้อความ] บันทึก Note — ใช้ 40 ตัวอักษรแรกเป็นหัวข้ออัตโนมัติ โน้ต ไอเดียโปรเจกต์ WMS ใหม่
memo [ข้อความ] เหมือน โน้ต — ใช้ภาษาอังกฤษแทนได้ memo review slides before meeting
ดูรายจ่าย แสดงสรุปรายจ่ายวันนี้ — จำนวนรายการ + ยอดรวม + 5 รายการล่าสุด ดูรายจ่าย
ดูงาน แสดงงานที่ยังไม่เสร็จ — To Do + In Progress ทั้งหมด (สูงสุด 5 รายการ) ดูงาน
help แสดงคำสั่งทั้งหมดผ่าน LINE — ใช้ ช่วย หรือ คำสั่ง แทนได้ help

⚙️ ตั้งค่า LINE Bot 3 ขั้นตอน ▼
1
ตั้ง LINE_TOKEN ใน Apps Script
เปิด script.google.com → เลือก Project → ⚙️ Project Settings → Script Properties
เพิ่ม Property: LINE_TOKEN = <Channel Access Token>

หา Token ได้ที่: LINE Developers → Channel → Messaging API → Channel access token
2
ตั้ง Webhook URL ใน LINE Developers
เปิด developers.line.biz → เลือก Channel → Messaging API → Webhook settings
ใส่ URL:
Apps Script Webhook URL
https://script.google.com/macros/s/AKfycbzgn1rNxQ9MB-65dxuj-68c_MIbOHtnwBKXI-VSx54YM50fZ8t162r2Isb6UIcrdYXAoA/exec
เปิด Use webhook แล้วกด Verify — ต้องได้ ✅ Success
3
ปิด Auto-reply (แนะนำ)
เปิด manager.line.biz → เลือก Account → Chat settings
ปิด Auto-reply messages และ Greeting messages
เพื่อป้องกัน Bot ตอบซ้ำเมื่อรับ Webhook
